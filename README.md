# Prediction_ventes  

Prédiction des ventes à l’aide d’un modèle de réseau de neurones dans un Jupyter Notebook.

## Vue d’ensemble  

Ce projet Jupyter Notebook se concentre sur la prédiction des ventes à l’aide d’un modèle de réseau de neurones.

## Prérequis  

- Jupyter Notebook  
- Python 3.x  
- TensorFlow  
- NumPy  
- Pandas  
- Scikit-learn  

## Données  

Placez votre fichier de données (`ventes.csv`) dans le même dossier que le Jupyter Notebook.

## Utilisation  

Ouvrez le Jupyter Notebook (`ventes_prediction.ipynb`) et exécutez les cellules dans l’ordre.
